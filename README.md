# Clone and Install

``` bash
git clone https://github.com/Moniruzzaman2525/Develop-Me.git
cd devlop-portfolio
npm install
# or
npm install -f

```

# Run development server
``` bash
npm run dev
``

Open [http://localhost:3000](http://localhost:3000)


### Build for Production

``` bash
npm run start
```
